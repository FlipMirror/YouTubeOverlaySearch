Расширение для автоматического поиска видео YouTube по запросу из адресной строки поисковой системы Яндекс и Google, результат которого выводится в окне расширения и запуска в отдельном Overlay плеере поверх страницы.   
Доступные функции: результаты поиска в окне расширения, запоминание положения скролла/прокручивания списка видео, проигрывание видео поверх страницы, просмотр комментариев.

Установка:

1. Скачать расширение:

* [Скачать YouTube Overlay Search ZIP](https://github.com/FlipMirror/YouTubeOverlaySearch/raw/refs/heads/main/YouTube%20Overlay%20Search/downloads/YouTube%20Overlay%20Search.zip)

2. Войти в раздел "Расширения" и "Управление расширениями" браузера. Передвинуть ползунок "Режим разработчика". Нажать "Загрузить распакованное расширение" и выбрать папку с необходимым расширением.
